# Contributor Covenant Code of Conduct

This project adopts the [Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

Please read the full text at the link above. It covers:

- **Our Pledge** — fostering an open, welcoming, and inclusive community
- **Our Standards** — expected and unacceptable behaviors
- **Enforcement Responsibilities** — community leader obligations
- **Scope** — where this code of conduct applies
- **Enforcement** — how instances are handled
- **Enforcement Guidelines** — consequence ladder
- **Attribution** — credit to the Contributor Covenant

## Reporting

Instances of behavior that violates this code of conduct may be reported to the project maintainers. All complaints will be reviewed and investigated promptly and fairly.

## Attribution

This Code of Conduct is adapted from the [Contributor Covenant](https://www.contributor-covenant.org), version 2.1, available at <https://www.contributor-covenant.org/version/2/1/code_of_conduct.html>.
