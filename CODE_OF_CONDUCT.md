# Code of Conduct

This project follows the [Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

All contributors, maintainers, and participants are expected to uphold a welcoming, inclusive, and respectful environment.

## Reporting

If you experience or witness unacceptable behavior, please report it by contacting the project maintainer. All reports will be handled with discretion.

## Enforcement

Project maintainers are responsible for clarifying and enforcing standards of acceptable behavior and will take appropriate and fair corrective action in response to any behavior they deem inappropriate.

## Attribution

This Code of Conduct is adapted from the [Contributor Covenant](https://www.contributor-covenant.org), version 2.1.
